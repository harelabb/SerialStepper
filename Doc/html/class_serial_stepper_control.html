Detailed Description

template<typename UINT, byte MAX_STEPPERS = 2 * sizeof(UINT)>
class SerialStepperControl< UINT, MAX_STEPPERS >

Intermediate abstract base class for StepperControl.

This class takes care of the communication between the Stepper and the StepperControl, leaving only two funtions to be implemented at the leaf class;
doBegin(), that initializes the hardware, and
doMove() that does the real physical output to the stepper motors.

Parameters
UINTunsigned integral type to store all ouput values. One byte is needed for every 2 Stepper.
MAX_STEPPERThe maxomum number of Stepper motors the StepperControl can handle. Defaults to two times the number of bytes in UINT.
